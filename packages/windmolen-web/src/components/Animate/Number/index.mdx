# Animate.Number

Animate.Number uses https://www.npmjs.com/package/react-countup for making the
animation available. All the non base props such as `className` will be passed
to the `<CountUp />` component. For more info about available props, visit
https://www.npmjs.com/package/react-countup

## Example
Take note that you normally would not have to specify "scrollableParent"
property. The default is "window" which should be good, unless your main
scrollable element is not "body".
<Playground wrapper={Wrapper}>
  <div style={{ height: '100vh', width: '100%', textAlign: 'center', fontWeight: 'bold' }}>Keep scrolling</div>
  <div style={{ height: '100vh', width: '100%', textAlign: 'center', fontWeight: 'bold' }}>Keep scrolling</div>
  <H1>
    <Animate.Number
      scrollableParentSelector=".css-d9u7f0-Wrapper"
      renderedItemProps={{
        end: 100,
        suffix: '%',
      }}
    />
  </H1>
</Playground>
