# Checkbox

The checkbox component is a wrapper around the `SwitchBase` component.

## SwitchBase Props
<PropsTable of={SwitchBase} />

## Props
<PropsTable of={Checkbox} />

## Basic usage
<Playground wrapper={Wrapper}>
  <Checkbox
    onChange={(checked) => {
      console.log('Basic usage - onChange', checked)
    }}
  />
</Playground>

## With label
<Playground wrapper={Wrapper}>
  <Checkbox label="Gender" value="my value" />
</Playground>

## With label on the left
<Playground wrapper={Wrapper}>
  <Checkbox label="Gender" labelPlacement="start" value="my value" />
</Playground>

## Custom icons
<Playground wrapper={Wrapper}>
  <div>
    <Checkbox
      label="This will switch icon only"
      value="my value"
      icon="twitter"
      iconChecked="facebook"
      iconVariant={2}
    />
  </div>

  <div>
    <Checkbox
      label="This will switch variant"
      value="my value"
      icon="twitter"
      iconVariant={2}
      iconVariantChecked={0}
    />
  </div>

  <div>
    <Checkbox
      label="This will switch icon and variant"
      value="my value"
      icon="twitter"
      iconChecked="facebook"
      iconVariant={2}
      iconVariantChecked={0}
    />
  </div>
</Playground>

## Using variants
Note that there is a `style` attribute being passed with margin, which is used
just for the sake of an example. Spacing and other additional styling should be
provided in the webapp.
<Playground wrapper={Wrapper}>
  <Checkbox
    label="1"
    value="people1"
    icon="people--one"
    variant="button-large"
    iconFontSize="170%"
    desktopIconFontSize="300%"
  />
