## Using variants
Note that there is a `style` attribute being passed with margin, which is used
just for the sake of a decent looking example. Spacing and other additional
styling should be applied in the webapp.

The `RadioGroup` has props that will be passed to the child `Radio` components.
Check out the `Props` section at the `RadioGroup` component.

<Playground wrapper={Wrapper}>
  <div>
    <H3>button-large</H3>
    <RadioGroup
      name="gender"
      value="people1"
      onChange={(value) => {
        console.log('Using variants - onChange', value);
      }}
      variant="button-large"
      iconFontSize="60px"
    >
      <Radio
        label="1"
        value="people1"
        icon="people--one"
      />
      <Radio
        label="2"
        value="people2"
        icon="people--two"
      />
      <Radio
        label="3"
        value="people3"
        icon="people--three"
      />
    </RadioGroup>
  </div>

  <div style={{ marginTop: '60px' }}>
    <H3>button-small</H3>
    <RadioGroup
      name="gender"
      value="people2"
      onChange={(value) => {
        console.log('Using variants - onChange', value);
      }}
      variant="button-small"

      <Radio
        label="1"
        value="people1"

      />
