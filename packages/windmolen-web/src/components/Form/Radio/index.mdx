# Radio

The Radio component is a wrapper around the `SwitchBase` component.

<PropsTable of={SwitchBase} />

<PropsTable of={Radio} />

## Basic usage
<Playground wrapper={Wrapper}>
  <Radio
    value="my value"
    onChange={(checked) => {
      console.log('Basic usage - onChange', checked)
    }}
  />
</Playground>

<Playground wrapper={Wrapper}>

</Playground>

## Custom icons
<Playground wrapper={Wrapper}>
  <div>
    <Radio
      label="This will switch icon only"
      value="my value"
      icon="twitter"
      iconChecked="facebook"
      iconVariant={2}
    />
  </div>

  <div>
    <Radio
      label="This will switch variant"
      value="my value"
      icon="twitter"
      iconVariant={2}
      iconVariantChecked={0}
    />
  </div>

  <div>
    <Radio
      label="This will switch icon and variant"
      value="my value"
      icon="twitter"
      iconChecked="facebook"
      iconVariant={2}
      iconVariantChecked={0}
    />
  </div>
</Playground>

## Using variants
Note that there is a `style` attribute being passed with margin, which is used
just for the sake of an example. Spacing and other additional styling should be
provided in the webapp.
<Playground wrapper={Wrapper}>
  <RadioGroup
    name="gender"
    value="people1"
    onChange={(value) => {
      console.log('Using variants - onChange', value);
    }}
    variant="button-large"
    iconFontSize="60px"
  >
    <Radio
      label="1"
      value="people1"
      icon="people--one"
      style={{ margin: '20px' }}
    />
    <Radio
      label="2"
      value="people2"
      icon="people--two"
      style={{ margin: '20px' }}
    />
    <Radio
      label="3"
      value="people3"
      icon="people--three"
      style={{ margin: '20px' }}
    />
  </RadioGroup>
</Playground>
