# ComponentGroup

A component group is used to add spacing in-between blocks. Component get
separated by default with a "large" spacing. Components inside a ComponentGroup
get separated by default with a "medium" spacing.

## Props
<PropsTable of={ComponentGroup} />

## Example
<Playground wrapper={Wrapper}>
  <ComponentGroup padding="large">
    <section>
      <H2>Lorem ipsum</H2>
    </section>
    <section>
      <H2>Lorem ipsum spaced</H2>
    </section>
  </ComponentGroup>

  <ComponentGroup
    padding="large"
    separator="border"
    separatorPlacement="top"
  >
    <section>
      <H2>Lorem ipsum</H2>
    </section>
  </ComponentGroup>
</Playground>
