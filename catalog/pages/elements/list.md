### Normal list
```react
<List items={['Item 1', 'Item 2']} />
```

### Unstyled list
```react
<List
  items={['Item 1', 'Item 2']}
  unstyled
/>
```

### API

```table
columns:
  - Property
  - Description
  - Type
  - Default
rows:
  - Property: items
    Description: list items
    Type: array
  - Property: unstyled
    Description: renders list without the bullets
    Type: bool
```
